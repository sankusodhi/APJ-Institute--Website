🔧 CUSTOMIZATION QUICK TIPS:

Change Colors:
→ Edit Tailwind classes: from-blue-600 → from-purple-600

Update Navigation:
→ Edit navLinks array in Navbar.jsx

Add Logo:
→ Replace text with <img> tag

Change Styling:
→ Modify Tailwind classes (h-16, px-4, etc.)

Hide on Mobile:
→ Add class: hidden md:block

Show Only on Mobile:
→ Add class: md:hidden
